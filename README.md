# Tokobekas

Tokobekas is a web platform for buying and selling second-hand goods, designed to provide a seamless, secure experience for both buyers and sellers. Built with PHP, Bootstrap, and MySQL, Tokobekas streamlines the process of listing, browsing, and transacting used items online.

## Features

- **User Authentication:** Register and log in securely to manage your listings.
- **Product Listings:** Add, view, and browse used items with images and descriptions.
- **User Dashboard:** Track your listings and engage with community testimonials.
- **Responsive Design:** Modern, mobile-friendly interface powered by Bootstrap 5 and custom CSS/SCSS.
- **Community Testimonials:** Platform highlights experiences and feedback from users and developers.
- **Easy Transactions:** Facilitates direct deals between sellers and buyers without extra fees.

## Technologies Used

- **PHP:** Backend logic, session management, database operations (PDO + MySQL).
- **Bootstrap 5:** Responsive UI framework.
- **SCSS / CSS:** Custom styles for branding and layout.
- **JavaScript:** Interactive elements (sliders, quantity controls, password toggling).
- **HTML:** Page structure.
- **MySQL:** Persistent storage for users and items.

## Getting Started

> _Installation instructions not provided. Please configure your PHP server and connect to a MySQL database as referenced in `app/core/Database.php`._

## Project Structure

- `/app/view/` : Main page templates (login, register, dashboard, item list)
- `/app/core/Database.php` : MySQL database connection handler
- `/public/css/`, `/public/scss/` : Stylesheets
- `/public/js/` : JavaScript for interactivity
- `/public/images/` : Branding and product images
